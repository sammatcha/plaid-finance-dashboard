# Plaid Finance Dashboard

A monorepo skeleton for a personal finance dashboard powered by [Plaid](https://plaid.com/). Includes an Express + TypeScript backend and a Vite + React + TypeScript frontend.


## Structure

- `backend/` — Node.js + Express + TypeScript API
- `frontend/` — Vite + React + TypeScript app
- `docker-compose.yml` — local Postgres 16 instance for development

## Getting Started

```bash
npm install
docker-compose up -d
npm run dev
```
